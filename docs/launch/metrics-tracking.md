## Action triggers

| Signal | Action |
|---|---|
| HF downloads > 100 with 0 Gumroad sales | Add a clearer CTA / pricing block to dataset card |
| GH stars growing but 0 sales | Check if Gumroad links are discoverable |
| Reddit post <20 upvotes | Try a different subreddit or post timing |
| Gumroad Individual sales > 5 | Promote Commercial licence more actively |
| Any negative HF review | Respond within 48 h and open a tracking issue |
